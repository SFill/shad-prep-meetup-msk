# Сравнение цен

Подготовительные курсы в магистратуру ФКН — https://fdp.hse.ru/mag_courses/ — 40 тысяч рублей за семестр, в котором 22 занятия, включая внутренние итоговые экзамены. Получается 1800 рублей за занятие.
